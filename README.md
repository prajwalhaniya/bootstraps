# bootstraps

## node project

Create a node project using the below command

```
bash <(curl -s https://raw.githubusercontent.com/prajwalhaniya/bootstraps/refs/heads/master/bootstrap.sh) PROJECT_NAME TEMPLATE_TYPE
```
